std::chars_format

Defined in header <utility> 
enum class chars_format {
    scientific = /*unspecified*/,
    fixed = /*unspecified*/,
    hex = /*unspecified*/,
    general = fixed | scientific
};
 (since C++17)

A BitmaskType used to specify floating-point formatting for std::to_chars and std::from_chars.
