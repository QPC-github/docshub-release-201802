WebGLRenderingContext.getActiveUniform

The WebGLRenderingContext.getActiveUniform() method of the WebGL API returns a WebGLActiveInfo object containing size, type, and name of a uniform attribute at a given position in a WebGLProgram.

Syntax

WebGLActiveInfo = WebGLRenderingContext.getActiveUniform(program, location);

Parameters

program
A WebGLProgram specifying the WebGL shader program from which to obtain the uniform variable's information.
location
A GLuint specifying the index of the uniform attribute to get. This value is returned by getUniformLocation().

Return value

A WebGLActiveInfo object describing the uniform.

Examples

gl.getActiveUniform(program, i);
