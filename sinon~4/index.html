Compatibility

ES5.1

Sinon v4.2.2 is written as ES5.1 and requires no transpiler or polyfills to run in the runtimes listed below.

Supported runtimes

v4.2.2 has been verified in these runtimes:

  • Firefox 45
  • Chrome 48
  • Internet Explorer 11
  • Edge 14
  • Safari 9
  • Node 4

There should not be any issues with using Sinon v4.2.2 in newer versions of the same runtimes.

If you need to support very old runtimes that have incomplete support for ES5.1 you might get away with using loading es5-shim in your test environment. If that fails, we recommend getting a legacy releases of Sinon.
