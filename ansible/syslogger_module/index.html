syslogger - Log messages in the syslog

New in version 2.4.

  • Synopsis
  • Options
  • Examples
    • Status

Synopsis

  • Uses syslog to add log entries to the host.
  • Can specify facility and priority.

Examples

# Full example
- name: Test syslog
  syslogger:
    msg: "Hello from ansible"
    priority: "err"
    facility: "daemon"
    log_pid: true

# Basic usage
- name: Simple Usage
  syslogger:
    msg: "I will end up as daemon.info"
