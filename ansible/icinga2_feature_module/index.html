icinga2_feature - Manage Icinga2 feature

New in version 2.3.

  • Synopsis
  • Options
  • Examples
    • Status

Synopsis

  • Enable or disable an Icinga2 feature

Options

parameter required default choices comments
name
 yes
This is the feature name to enable or disable.
state
 no present
  • present
  • absent
Apply feature state.

Examples

- name: Enable ido-pgsql feature
  icinga2_feature:
    name: ido-pgsql
    state: present

Status
